## Instação

### Clone o repositório

```
git clone
```

### Acesse o diretorio

```
cd <nome-da-pasta>
```

### Instale as dependências

```
npm install
```

### Inicie a aplicação

```
npm run dev
```

### A aplicação, por padrão, fica na porta:

```
http://localhost:3000/
```
